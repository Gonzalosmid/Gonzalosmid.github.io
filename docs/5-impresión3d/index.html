5-Impresión 3D

Impresión 3D:

Para este encargo se debía realizar el proceso previo a una impresión 3D convencional. Para ello se utilizó un software llamado UltimateCura, en el cual se pueden establecer los parámetros que la impresora ejecutará posteriormente, por ejemplo, el material con el cual se creará la figura, el tamaño y la altura de las capas, para darles un acabado y definición distinto.

Este tipo de sistemas nos otorga algo muy importante a la hora de realizar impresiones 3D, el tiempo estimado que demora la construcción de la pieza, en función de la calidad que se espera.

  • Link descarga UltimateCura

https://ultimaker.com/es/software/ultimaker-cura

Paso 1

El primer paso consiste en generar un modelo en formato .stl o bien bajarlo de una plataforma open source. Para la realización del tutorial utilizaré el archivo adjunto.

  • Link modelo

https://www.thingiverse.com/thing:4902471

Elección de la impresora:

El programa tiene instalado una serie de impresoras con las cuales podemos configurar la plataforma. Para este caso usaremos una impresora Creality Ender 3, este modelo es ampliamente conocido por ser una gran opción calidad/precio a la hora de buscar una. En Chile, podemos encontrarla por aproximadamente $200000 pesos.

Elección del material:

El material que usamos para este tutorial es un PLA + de la marca eSun. Es importante que sea de 1.75 mm.

Boquilla:

La boquilla que conforma un término medio entre una apertura muy amplia y otras de menor diámetro, es la boquilla de 0,4 mm.

Definir los parámetros:

Para imprimir una pieza como esta no es necesario tener una mala definición, puesto que es pequeña y su nivel de detalle requiere cierta lentitud y fineza, por lo que el alto de capa se ajusta a 0,2 mm.
